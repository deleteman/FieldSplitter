FieldSplitter by deleteman

JQuery plugin that turns a single input field into a multi-field input

This jQuery plugin splits an input field into several input fields that work toghether jumping from one field into the next one.

Dependencies

jQuery 1.6.x

Install

Just add the script tag after including jQuery.
